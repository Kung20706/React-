# todo_app

# 關鍵字

- golang
- beego
- react
- material-ui

# 說明

此專案練習建構待辦清單網站，後端以 beego建構，前端則以 react建構。利用 material-ui的 Grid system製作 RWD響應式網站。

# 安裝

STEP 1. 專案 clone至 GOPATH的 src目錄底下

STEP 2. [安裝 beego套件](https://beego.me/)

STEP 3. [安裝 Node.js](https://nodejs.org/en/)

# 附錄

## 測試執行後端
切換至 todo_app/serivce/
```sh
$ cd service
```

啟動後端服務
```sh
$ bee run
```

## 安裝前端相關模組
切換至 todo_app/web/
```sh
$ cd web
```

啟動前端服務
```sh
$ npm install
```


## 測試執行前端
切換至 todo_app/web/
```sh
$ cd web
```

啟動前端服務
```sh
$ npm run start
```
